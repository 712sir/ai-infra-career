## 01-hash — 哈希（3题）

| # | 题号 | 题目 | 难度 |

| 1 | 1 | 两数之和 | 🟢 |
| 2 | 49 | 字母异位词分组 | 🟡 |

### 章节总结

## 15-dp-2d — 多维动态规划（5题）

| # | 题号 | 题目 | 难度 |
|---|------|------|------|
| 1 | 62 | 不同路径 | 🟡 |
| 2 | 64 | 最小路径和 | 🟡 |
| 3 | 5 | 最长回文子串 | 🟡 |
| 4 | 1143 | 最长公共子序列 | 🟡 |
| 5 | 72 | 编辑距离 | 🔴 |

### 章节总结
- 网格路径 = 二维DP入门，dp[i][j] = dp[i-1][j] + dp[i][j-1]
- 编辑距离面试极高频率，dp[i][j] 表示 word1[0:i] → word2[0:j] 的最少操作数
