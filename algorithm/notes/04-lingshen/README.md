## 如何与前面题单衔接

灵神题单 = 在你刷完代码随想录 + Hot 100 + 面试经典150 之后，用难度分排序来做**螺旋上升**

具体操作：
1. 选定一个专题（如 DP）
2. 按难度分从低到高刷
3. 刷 10 题后切换到另一个专题（如二分）
4. 循环往复，逐渐提升难度分
