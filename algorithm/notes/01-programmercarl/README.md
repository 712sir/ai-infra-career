## 07-monotonic-stack — 单调栈

| # | 题号 | 题目 | 核心技巧 |
|---|------|------|----------|
| 1 | 739 | 每日温度 | 单调递减栈（存下标） |
| 2 | 496 | 下一个更大元素I | 单调栈 + map |
| 3 | 503 | 下一个更大元素II | 循环数组 = 遍历两遍 |
| 4 | 42 | 接雨水 | 单调栈 / 双指针 / DP |
| 5 | 84 | 柱状图中最大的矩形 | 单调递增栈 + 哨兵 |

### 章节总结
- **单调栈本质**：找到每个元素**左边/右边第一个比它大/小的元素**
- **递增栈**：栈内元素递增，弹出时遇到右边第一个更小的 → 适合找"最大矩形"
- **递减栈**：栈内元素递减，弹出时遇到右边第一个更大的 → 适合找"下一个更大元素"
- **技巧**：存下标不存值、循环数组遍历两遍、加哨兵简化边界

- **01背包 vs 完全背包**：01背包倒序遍历背包容量，完全背包正序
- **组合 vs 排列**：外层物品内层容量 = 组合数，外层容量内层物品 = 排列数
- **股票问题**：状态定义是关键（持有/不持有/第k次/冷冻期）
- **回文问题**：647→5→516→132 层层递进；647 的 `isPalindromic[i][j]` 是 5 和 132 的基础
